# 🐍 Data Analyzer and Transformer

A menu-driven Python program that analyzes and transforms data using various Python functions and concepts.

---

## 📌 About the Project

This project demonstrates core Python programming concepts through a hands-on Data Analyzer and Transformer program. Users can input data and perform various operations through an interactive console menu.

---

## ✨ Features

| Menu Option | Feature | Concept Used |
|---|---|---|
| 1 | Input 1D Array | List, map() |
| 2 | Display Data Summary | Built-in Functions |
| 3 | Factorial / Fibonacci | Recursion |
| 4 | Filter by Threshold | Lambda + filter() + map() |
| 5 | Sort Data | Sorting |
| 6 | Dataset Statistics | Return Multiple Values |
| 7 | Duplicates & Unique Values | User-defined Functions |
| 8 | *args & **kwargs Demo | *args, **kwargs, __doc__ |
| 9 | 2D Array Input & Display | Nested Lists / 2D Array |
| 10 | Exit | — |

---

## 🧠 Concepts Demonstrated

- **Built-in Functions** — `len()`, `min()`, `max()`, `sum()`
- **User-defined Functions (UDF)** — `find_duplicates()`, `find_unique()`, `statis()`
- **`*args` and `**kwargs`** — accept multiple values and print key-value pairs
- **`__doc__` strings** — every function has a docstring
- **Recursion** — factorial and Fibonacci sequence
- **Lambda Functions** — used with `filter()` and `map()`
- **Global Keyword** — `global_total` and `global_mean` shared across functions
- **Return Multiple Values** — `statis()` returns min, max, total, average
- **1D and 2D Arrays** — input, display, and grid format
- **Sorting** — ascending and descending using `sorted()`
